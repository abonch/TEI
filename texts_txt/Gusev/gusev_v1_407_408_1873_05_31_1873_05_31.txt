Гусев Николай Николаевич
«Летопись жизни и творчества Л.Н. Толстого»
31 мая 1873 года

Т. сообщает H. Н. Страхову, что он «придумал», ради лучшего распространения «Азбуки», «расшить» ее «на 12 маленьких книжечек» и «разослать по земствам». Заглавия книжечек: «Азбука. Книга I»; «Первая русская книга для чтения. Книга II»; «Первая славянская книга для чтения. Книга III»; «Вторая русская книга для чтения. Книга IV»; «Вторая славянская книга для чтения. Книга V»; «Третья русская книга для чтения. Книга VI»; «Третья славянская книга для чтения. Книга VII»; «Четвертая русская книга для чтения. Книга VIII»; «Четвертая славянская книга для чтения. Книга IX»; . «Арифметика. Четыре правила. Книга X»; «Арифметика. Дроби. Книга XI»; «Руководство для учителя. Книга XII». Материал всех книжек целиком совпадает с материалом «Азбуки».
